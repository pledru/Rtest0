## Instalation

### Dependencies

Most dependencies can be installed along with Capybara through CRAN.

### Install the package
Install devtools:
```r
install.packages("devtools")
```
Install the package from GitHub:
```r
library("devtools")
install_github("pledru/Rtest0")
```

```r
library("Rtest0")
